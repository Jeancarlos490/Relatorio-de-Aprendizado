# Validação Basica

Verificar a idade e o sexo escolhido no cadastro

---

## O que foi aprendido
- 1. Conceito aprendido: If/else
- 2. Conceito aprendido: DOM
- 3. Conceito aprendido: Switch
- 4. Conceito aprendido: Operadores lógicos

> No desafio/atividade atual teve o objetivo de aprender conceitos basicos
na linguagem javascript como o modelo DOM e uso de operadores lógicos.

---

##  Tecnologias usadas
- Linguagem de hypertexto: HTML
- Linguagem de programação: Javascript
- Ferramenta de edição usada: VS code com Live-Server

---

## Objetivo do projeto
Iniciar a aprender a linguagem javascript com um pequeno desafio que tem
como objetivo simular alguns conceitos para validação de um cadastro.
