# Controle de Gastos

Anotar os gastos permanentes e gastos variáveis.

---

## O que foi aprendido
- 1. Conceito aprendido: Uso e manipulação de Arrays;
- 2. Conceito aprendido: Criação e exclusão de elementos HTML via javascript;
- 3. Conceito aprendido: uso do elemento javascript "for".

> O desafio/atividade atual teve o objetivo de aprender sobre a manipulação
de listas denominadas Arrays usando Javascript.

---

##  Tecnologias usadas
- Linguagem de hypertexto: HTML
- Linguagem de estilização: CSS
- Linguagem de programação: Javascript
- Ferramenta de edição usada: VS code com Live-Server

---

## Objetivo do projeto
Criar um controlador de gastos que tem como finalidade para o usuario
anotar os seus gastos ao longo do tempo, alimentando a lista para
observar os gastos com o tempo.
