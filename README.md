# Relatório de Estudos

Este repositório tem como objetivo documentar minha evolução prática em programação,
registrando estudos, exercícios e desafios realizados ao longo do tempo.

Não se trata de um portfólio de projetos finais, mas de um histórico técnico de aprendizado
contínuo e consolidação de fundamentos.

---

## Objetivo

- Registrar a evolução gradual dos estudos
- Praticar conceitos fundamentais de programação
- Aplicar lógica em exercícios e desafios autorais
- Manter um histórico real e verificável de aprendizado

---

## Organização do Repositório

A estrutura do repositório é organizada por temas e/ou etapas de estudo.
Cada pasta representa um conjunto de conceitos praticados em determinado período.

O método utilizado para a organização das pastas será separar cada linguagem de forma individual,
sendo que cada uma terá seu próprio código e um README explicando o que foi aprendido,
de forma concreta, com comentários no próprio código.
